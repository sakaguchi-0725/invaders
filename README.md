# Invaders - スペースインベーダー

## プロジェクト概要

このリポジトリは、**スペースインベーダー風シューティングゲームの実装を通じて、ゲーム開発の基礎を学習するための学習用リポジトリ**です。

**目的**: ゲームプログラミングの基本概念を学習・実践する
- ゲームループアーキテクチャ（Update/Drawサイクル）
- スプライトレンダリングとアニメーション
- 衝突判定
- 入力処理
- ゲーム状態管理
- エンティティ管理

**技術スタック**:
- 言語: Go 1.25.3
- ゲームエンジン: Ebiten v2.9.7

## 実行方法

```bash
# ゲームを実行
go run main.go

# ビルド
go build -o invaders main.go
```

## 開発メモ

### AI解析用の設計方針
このプロジェクトはAIが解析・継続開発しやすい構造を目指しています:
- 関心の分離（エンティティ、システム、状態）
- 明確なディレクトリ構造
- 素材のクレジットとソースの明記
- 実装状況の追跡によるコンテキスト提供

### 学習目標
1. ゲームループパターンの理解
2. スプライトベースレンダリングの実装
3. ゲームエンティティとその相互作用の管理
4. リアルタイムアプリケーションでのユーザー入力処理
5. ゲーム物理と衝突判定の実装
6. ゲーム状態遷移の管理
